# Time Investment Advisers LLC Premium Website

This package contains a premium multi-page static website for GitHub Pages.

## Included
- Premium homepage redesign
- About, Services, Disclosures, and Contact pages
- Uploaded logo and headshot assets
- Email updated to timeinvestmentadvisers@gmail.com

## Assets
All images are in:
assets/images/

## Publish on GitHub Pages
1. Upload all files and folders to your GitHub repository root.
2. Make sure `index.html` is in the root.
3. In GitHub: Settings > Pages
4. Set source to `main` and folder to `/ (root)`

## Custom domain
If you use your domain:
- keep a file named `CNAME`
- it should contain only:
  www.timeinvestmentadvisers.com

## Before publishing
Review:
- disclosures
- fee wording
- registration references
- contact details
- ADV Part 2A link

# Time Investment Advisers LLC Website

This package contains a simple multi-page static website that can be hosted on GitHub Pages.

## Files
- index.html
- about.html
- services.html
- disclosures.html
- contact.html
- style.css

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In repository Settings > Pages, enable GitHub Pages from the main branch.
4. Add your custom domain if desired.

## Before publishing
Review and update:
- Contact information
- Disclosure language
- Fee language
- Any regulatory references
- ADV Part 2A link
